Q To-Do

Q To-Do is a simplistic todo list organizer for MeeGo/Harmattan and Nemo/Mer.

View the Project on GitHub ruedigergad/qtodo

About

Q To-Do is a simplistic todo organizer for MeeGo/Harmattan and Nemo/Mer. It features a clear, simplistic, and easy to use user interface that is optimized for multi-touch touch screens.

Usage

On the first start no entries will be present. To add new entries simply click "+" in the toolbar. Please note: there are two types of entries "todos" and "notes". Todos can have subentries, notes don't.

To navigate to sub-entries of a todo simply swipe from right to left. You should see the list move side-ways as you do it. Alternatively, you can click the small button on the right side of a todo entry. To navigate to the parent items simply swipe from left to right or click the arrow pointing to the left in the toolbar.

To add sub-entries to a new todo entry simply navigate to the list showing the sub-entries as described above. You will see an empty list. There you can then add new entries the same way you did before after Q To-Do was started for the first time.

A "progress" bar indicates the progress of todos containing other todos. See also the screenshots below.

To reorder entries use a two finger gesture. Select the entry you want to move, keep it pressed, and with a second finger move up or down.

Double-clicking an entry opens the edit field. Long-pressing opens a menu with more options.

Installation

Q To-Do is currently available for MeeGo/Harmattan and Nemo/Mer. The MeeGo version can be downloaded from app.formeego.org or Nokia Store.

The Nemo/Mer version is currently only available via my download repository. To add the repository and install Q To-Do open a shell and switch to the "root" user. As root type the following:

zypper ar http://repo.pub.meego.com/home:/wonko/CE_MW_MTF_CE_MW_Shared_armv7hl/home:wonko.repo
zypper ref
zypper up

Not that the repository is not signed yet. So you will be asked if you want to used an unsigned repository. Simply answer "yes" here.
